# ClosetCue 👗

ClosetCue is your **AI-inspired outfit assistant** that helps you find the perfect look based on your **weather, mood, and events**. No more decision fatigue — ClosetCue makes styling effortless and fun!

---

## 🔧 Tech Stack

Frontend: React, HTML, CSS, JavaScript  
AI Integration: Gemini API  
Deployment: Netlify

---

## 🚀 Getting Started

Follow these steps to run ClosetCue locally:

1. Clone the repository
   git clone https://github.com/your-username/ClosetCue.git

2. Navigate into the project folder
   cd ClosetCue

3. Install dependencies
   npm install

4. Start the development server
   npm start

Open http://localhost:5173 to view the app in your browser.

---

## 📌 Features

- **AI Outfit Recommendations:** Input your weather, mood, and event, and receive curated outfit suggestions.
- **Hero Section:** Catchy heading with a call-to-action button.
- **Responsive Layout:** Works beautifully on mobile, tablet, and desktop.
- **Interactive Cards:** Outfit suggestions with hover effects and details.
- **Clean Design:** Modern, minimal, and cozy styling.
